Lab 1

Abstract

This experiment examined the first order inverted pendulum, we obtain the numerical linearized transfer function model of the inverted, build the nonlinear system in simulink, and carry out a closed-loop simulation with the controller we designed.
Inverted pendulum control system (IPS) is a complex, unstable and nonlinear system, which is an ideal experimental platform for teaching control theory and conducting various control experiments. The typicality of inverted pendulum is that, as a controlled object, it is a high-order, unstable, multi-variable, nonlinear and strongly coupled complex controlled system, which can effectively reflect many problems in the control.
The research of inverted pendulum system can effectively reflect many typical problems in control, such as nonlinear problem, robustness problem, stabilization problem, follow-up problem and tracking problem. The control of inverted pendulum is used to test whether the new control method has strong ability to deal with nonlinear and unstable problems.

Introduction

The purpose of this experiment was to make inverted pendulum stable without upswing. These were be done by using linear transition, pole assignment and matlab simulink. We use our own simulink to simulate the control, and use the provided simulink to control the actual inverted pendulum, and compare the two results, get the differences, and then analyze the reasons, and expect to learn the stable control method for the inverted pendulum system through the above process.

Comment on any difference between the simulation and experimental results(for nonlinear)

It is obvious from the output images of $z$ that although the output results of both simulation and experiment can prove that the system is in the stable state. But the $z$ of simulation converge to 0 after a fluctuation, while the specific experiment is always in a fluctuation state——This is because during the simulation when the system gives an excitation (displacement) to the inverted pendulum and get a small offset angle, that is, the system is considered to have stabilized, so the motion is stopped and stabilized at a certain position, the $z$ is 0; but in the actual experiment when the system will always give excitation (displacement) to determine whether the system is stable, but because the calculated $[p_0 \ p_1 \ p_2 \ p_3 \ q_0 \ q_1 \ q_2 \ q_3]$ value is not perfect, as well as the influence of the surrounding environment, resulting in the system has not been able to really reach a stable state, so it has been in constant motion, in the hope of eventually reaching stability, so the angle and displacement will always be some range has been fluctuating.

The choice of $\alpha$ and its influence

First of all, we must first understand the impact of the value of $\alpha$ on $z$:
If $0 <\alpha < 1$, $z(t)$ will be the horizontal position of a point on the rod.
If $\alpha>1$, $z(t)$ will be the horizontal position of a point on the extended line of the rod.
In addition, if $\alpha$ = 1, $z(t)$ will be the horizontal position of the tip point of the rod and if $\alpha=0$, $z(t)$ will be the horizontal position of the cart.
For our lab2, the approximate range of is $[0.5,3]$, we finally choose 2.5 for $\alpha$’s value, but why: the obvious reason is when $\alpha=2.5$, the inverted pendulum can be easier to become stable and more stable for a long time.
And for deep understanding, the value of determines the influence of theta on the system—if it is too small, it proves that the relationship between the inverted pendulum system and theta is basically no, that is, whether the inverted pendulum is swinging has no effect, which will lead to the robustness of the system to be weak, because the output has no effect on the entire system; but if $\alpha$ is too large, if we want the $P_z(s)$ to become 0, then the x itself (that is, the displacement of the inverted pendulum) needs to be large, which can be regarded as a device that moves at the bottom requires a larger range of motion, but the moving range of the bottom mobile device of the provided inverted pendulum system is about 1m. If it is too large, $P_z(s)$ will not be equal to 0, that is, the entire system will not be stable.
